# sfo_parse
Quick and dirty PSV SFO parser


# Usage:
```
# Assumes 'param.sfo' in same directory as sfo_parse.py
./sfo_parse.py
# Alternatively specify path to param.sfo
./sfo_parse.py /path/to/param.sfo
```
